Tecnologias usadas:

Fastify (micro framework)
Função: Framework web para Node.js usado para criar APIs de forma rápida e performática.
Instalação:
npm install fastify

TypeScript
Função: Adiciona tipagem ao JavaScript para evitar erros e melhorar a organização do código.
Obs: O Node.js não reconhece TypeScript nativamente.
Instalação:
npm install -D typescript

Criar arquivo de configuração do TypeScript:
npx tsc --init

@types/node
Função: Permite que o TypeScript reconheça as APIs do Node.js (process, fs, path, etc).
Instalação:
npm install -D @types/node

TSX
Função: Permite executar arquivos TypeScript diretamente, sem precisar compilar antes.
Instalação:
npm install -D tsx

Uso exemplo:
npx tsx src/server.ts

ESLint
Função: Analisa o código e ajuda a evitar erros e más práticas.
Instalação:
npm install -D eslint @rocketseat/eslint-config

Migrations
Função: Histórico versionado das alterações no banco de dados (criação e mudança de tabelas).

Zod
Função: Valida os dados recebidos pela aplicação, garantindo que estejam no formato correto.

Vitest: 
Função: Ferramenta utilizada para realizar testes do nosso código
